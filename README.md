# Laravel-Checker
Tool to check Laravel website or not

# Usage

```
php laravel-site.php
```

# Response

Success :
```bash
1. https://site.com/ : {"laravel":true}
2. https://website.com/ : {"laravel":true}
```
Failed :
```bash
1. https://site.com/ : {"laravel":false}
2. https://website.com/ : {"laravel":false}
```
